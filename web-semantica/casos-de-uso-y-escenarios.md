# Casos de uso y escenarios

### Casos de uso de la capa semántica

* explotación analítica de datos clínicos en formato grafo,
* integración con herramientas semánticas externas,
* consultas de soporte a decisión para escenarios académicos,
* generación de capas interoperables para investigación.

### Escenarios prácticos documentados

#### Exportar conocimiento clínico

Uso real: entregar datos clínicos a un consumidor que entiende JSON-LD.

#### Consulta SPARQL manual

Uso real: tablero analítico de relaciones clínicas.

#### Pregunta en lenguaje natural

Uso real: usuarios sin conocimiento SPARQL obtienen respuestas semánticas.

#### Trazar una cita específica

Uso real: auditoría o análisis puntual de un evento clínico.
