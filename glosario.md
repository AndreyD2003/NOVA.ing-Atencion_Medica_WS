# Glosario

<details>

<summary><strong>OpenFeign</strong></summary>

Cliente HTTP declarativo usado para comunicación entre microservicios.

</details>

<details>

<summary><strong>DTO</strong></summary>

Objeto de transferencia para desacoplar contratos API del modelo persistente.

</details>

<details>

<summary><strong>Soft delete</strong></summary>

Eliminación lógica. El registro no se borra físicamente.

</details>

<details>

<summary><strong>IRI</strong></summary>

Identificador global de recursos en RDF.

</details>

<details>

<summary><strong>RDF</strong></summary>

Modelo de datos orientado a grafo basado en tripletas sujeto-predicado-objeto.

</details>

<details>

<summary><strong>OWL</strong></summary>

Lenguaje de ontologías para definir semántica e inferencias.

</details>

<details>

<summary><strong>SPARQL</strong></summary>

Lenguaje de consulta para grafos RDF.

</details>

<details>

<summary><strong>JSON-LD</strong></summary>

Serialización JSON para datos enlazados.

</details>

<details>

<summary><strong>Razonamiento OWL</strong></summary>

Inferencia de conocimiento derivado a partir de axiomas.

</details>

<details>

<summary><strong>Solape de citas</strong></summary>

Intersección de intervalos horarios para médico o paciente.

</details>

<details>

<summary><strong><code>create-drop</code></strong></summary>

Estrategia JPA que crea y elimina esquema al iniciar o detener la aplicación.

</details>

<details>

<summary><strong>Agregación de detalle</strong></summary>

Composición de datos desde varios microservicios en una sola respuesta.

</details>

<details>

<summary><strong>MSVC</strong></summary>

Microservicio con responsabilidad de negocio específica.

</details>
