# Clientes Feign

### Clientes declarados

* `PacienteClientRest`
* `MedicoClientRest`
* `DiagnosticoClientRest`

### `PacienteClientRest`

Se usa para obtener el detalle del paciente.

### `MedicoClientRest`

Se usa para obtener el detalle del médico.

### `DiagnosticoClientRest`

Se usa para listar diagnósticos por cita.

### Por qué existen

Permiten construir `CitaDetalle` y validar reglas del agendamiento.
